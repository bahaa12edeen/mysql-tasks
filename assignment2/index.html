1. Write a query to find the addresses (location_id, street_address, city, state_province, country_name) of all the departments.

ANSWER: • SELECT * FROM locations INNER JOIN countriesON locations.country_id=countries.country_id;

2. Write a query to find the name (first_name, last name), department ID and name of all the employees.

ANSWER: • SELECT employees.first_name, employees.last_name, departments.department_id FROM employees INNER JOIN departmentsON employees.department_id=departments.department_id;

3. Write a query to find the name (first_name, last_name), job, department ID and name of the employees who works in London.

ANSWER: • SELECT employees.first_name,employees.last_name, employees.job_id, department_id FROM employees WHERE country_id="LGW";

4. Write a SQL statement to rename the table countries to country_new.

ANSWER: • ALTER TABLE countries RENAME country_new;

5. Write a SQL statement to add a column region_id after state_province to the table locations.

ANSWER: • ALTER TABLE locations CHANGE region_id region_id INT(11) NOT NULL AFTER state_province;

6. Write a SQL statement to add a primary key for the columns location_id in the locations table.

ANSWER: • ALTER TABLE locations ADD PRIMARY KEY(location_id);

7. Write a query to display the names (first_name, last_name) using alias name “First Name", "Last Name"

ANSWER: • SELECT first_name AS FirstName, last_name AS LastName FROM employees WHERE 1;

8. Write a query to get all employee details from the employee table order by first name, descending.

ANSWER: • SELECT first_name FROM employees ORDER BY first_name DESC;
